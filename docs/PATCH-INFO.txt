Subject: [PATCH] hw/arm: Add Calypso SoC minimal machine

Add support for the Calypso SoC (TI ARM946-based) minimal machine.
This SoC was used in various old mobile phones.

Features:
- ARM946E-S CPU
- 256 KiB RAM (configurable)
- 4 MiB NOR Flash (CFI01)
- UART with basic I/O
- SPI controller stub
- Timer
- Multiple MMIO peripheral blocks

The machine supports loading ELF firmware and automatically patches
the flash pointer for compatibility with existing firmware images.
